How it works

From idea to running task in seconds

No YAML hell. No infrastructure headaches. Just describe what you need.

1

Describe in plain English

Use the web dashboard, CLI, or let an AI agent create tasks via MCP. Or write TypeScript/Python yourself — you choose the level of control.

2

Code in seconds, live instantly

AI generates validated task code — task.yaml + task.ts — and saves it to your tasks directory. Edit locally? Changes are live in ~100ms via filesystem watching. No build step, no deployment, no restart.

3

GitOps reconciliation

dicode watches all your sources — local directories, git repos, community stores — via TaskSet manifests. Content hashes, automatic diff, zero restarts. Like ArgoCD, but for automation tasks.

4

Test, monitor, iterate

Run tasks from the web dashboard or CLI. Watch live logs. Chain tasks together. Set up failure alerts. Review history. Test before production. All from one place.

True GitOps

Everything lives in git

A TaskSet manifest maps tasks from anywhere — your own repo, a community store, shared team repos — into one unified automation surface. Push a change, it's live in seconds. The same pattern as ArgoCD and Flux — but for automation tasks.

taskset.yaml 
apiVersion: dicode/v1
kind: TaskSet
metadata:
  name: my-infra

spec:
  entries:
    # Your own tasks
    pr-digest:
      ref: { path: ./tasks/pr-digest }

    # Community store task
    slack-alerts:
      ref: { url: github.com/dicode-community/tasks, path: slack-alerts }

    # Shared team repo
    deploy:
      ref: { url: github.com/my-org/shared-ops, path: deploy }

    # Nested TaskSet — compose hierarchies
    monitoring:
      ref: { path: ./monitoring/taskset.yaml }

tasks/pr-digest/task.yaml 
name: PR Digest
runtime: deno
cron: "0 9 * * 1-5"
params:
  repo: { type: string, default: "my-org/api" }
env:
  - GITHUB_TOKEN
  - SLACK_WEBHOOK
tasks/pr-digest/task.ts 
// That's it. Real code you can read, review, and debug.

const repo = await params.get("repo") ?? "my-org/api";
const resp = await fetch(
  `https://api.github.com/repos/${repo}/pulls?state=open`,
  { headers: { Authorization: `Bearer ${Deno.env.get("GITHUB_TOKEN")}` } }
);
const prs = await resp.json();

await fetch(Deno.env.get("SLACK_WEBHOOK"), {
  method: "POST",
  body: JSON.stringify({ text: `${prs.length} open PRs in ${repo}` })
});
